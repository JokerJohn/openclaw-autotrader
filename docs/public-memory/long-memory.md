# Trading Long Memory / 交易长期记忆

Updated / 更新时间: 2026-03-12 00:36:37 CST (UTC+08:00)

## good_exit_RIVN
**Lesson**: RIVN 最近的退出时点较有效，说明该类风险缩减纪律值得继续保留。

## cash_reserve_block
**Lesson**: 现金不足会直接阻断后续决策与轮动，必须保留至少20%机动现金。

## realtime_quote_gate
**Lesson**: 需要实时行情或明确的开盘状态才能进入实盘决策，缺行情时不要强行交易。

## fx_split_retry
**Lesson**: 换汇失败时应优先分批换汇并确认目标币种已到账，再继续股票下单。

## weak_followthrough_buy_RIVN
**Lesson**: RIVN 最近买入后的延续较弱，除非 EV 和胜率显著改善，否则不要继续追高。
