# Public Operating Strategy / 公开操作策略

This repository discloses the public operating principles of the OpenClaw trading workflow.
这个仓库公开 OpenClaw 交易流程的对外可披露原则。

## Core Rules / 核心规则

- Starting pocket capital / 起始口袋资金: `5000 HKD`
- Default market / 默认市场: `US` equities first / 以 `US` 市场为主
- Whitelist only / 仅限白名单
- No leverage / 不加杠杆
- No short selling / 不做空
- Continuous monitoring, bounded live execution / 持续监控，有限实盘执行

## Public Disclosure Boundary / 公开披露边界

- publish holdings status and trade direction / 公开持仓状态和交易方向
- publish timing rationale and no-trade rationale / 公开交易时机理由与观望理由
- do not publish code, secrets, raw logs, or private broker data / 不公开代码、密钥、原始日志和私有 broker 数据
